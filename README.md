# VotingDapp-去中心化的投票系统

## Getting Started

`npm run dev`

## 应用目的

创建一个去中心化的投票系统，允许用户在区块链上匿名投票。该系统需满足以下要求：
用户能够创建投票。  
用户能够参与投票。  
投票结果是公开透明的。

## 合约开发

使用 `solidity` 部署在 `sepolia` 测试链
合约地址:`0xEF9a9493E0312CbF08fa999b4B7570af86554A44`

## 前端 React 项目

`RainbowKit + wagmi + Next.js + NextUI`
